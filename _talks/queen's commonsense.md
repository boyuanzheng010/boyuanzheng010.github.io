Conducted research of text generation based on inferential atlas, supervised by Prof.Xiaodan Zhu.

Project details:
* Built inferential commonsense atlas based text generation system.
* Applied incremental inference in generation model to enhance generation coherence and interpretability.
* Utilized generation model to predict event influences and use influences to enhance dialogue system
performance.
* Surveyed into human’s reaction to events.

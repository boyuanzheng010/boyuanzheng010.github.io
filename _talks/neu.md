Conducted research in Machine Reading Comprehension, supervised by Prof.Feiliang Ren.

Project details:
* Designed text filter to select texts more relevant to the answer to improve the model preformance.
* Researched into text similarity algorithms(BLEU-4, Bm25 etc.) for text filter.
* Used BERT pre-trained model to generate better word representation.
* Participated in a Chinese Machine Reading Comprehension Competition and rank 11th.

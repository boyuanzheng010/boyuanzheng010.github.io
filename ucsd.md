Conducted research of Text Summarization, supervised by Prof.Julian McAuley.
Improve Text Summarization performance by incorporating personal information about the summarizer.

Project details:
* Investigated into text summarization data with personalized information.
* Designed filters to automatically select valid instance for text summarization.
* Built 3 Text summarization models enhanced by personalized information.
